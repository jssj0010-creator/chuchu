
# Luxe Theme Pack (Regional Pages)

이 패키지는 **서울/경기/인천 서브페이지**(예: `seoul.html`, `gyeonggi.html`, `incheon.html`)에
메인과 동일한 **고급/세련(Luxe Dark)** 테마를 적용하기 위한 공용 CSS를 제공합니다.

## 구성
- `styles/luxe.css`

## 적용 방법 (각 지역 페이지 공통)
1) 리포지토리 루트에 이 폴더 구조 그대로 업로드:
   - `/styles/luxe.css`
2) 각 지역 페이지의 `<head>` 끝부분( `</head>` 바로 위 )에 아래 줄 추가:
```html
<link rel="stylesheet" href="/styles/luxe.css">
```
3) 저장/커밋 후 새로고침.

### 참고
- 페이지마다 기존 인라인 스타일이 있어도, **luxe.css**가 *뒤에서* 로드되기 때문에 자연스럽게 우선 적용됩니다.
- 카드/버튼/배지/가격표/공지 박스 등의 클래스를 기존 페이지에서도 그대로 사용하면 스타일이 통일됩니다.
- 필요 시 추가로 커스터마이즈하고 싶다면 `luxe.css` 맨 아래에 덮어쓰기 규칙을 더 작성하세요.
